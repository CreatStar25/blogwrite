1. OpenClaw 是什么？

OpenClaw（曾用名 ClawdBot 或 MoltBot）是一款开源的本地化个人 AI 智能体（Agent）框架。它以一种永远在线的后台守护进程（Daemon）形式运行在你的设备上。

通过极简的内核和代理编程（Agentic Programming）理念，OpenClaw 可以直接接入你常用的通讯软件（如 Telegram、Discord、WhatsApp、Slack 等）并接受自然语言指令。它不仅擅长读写代码、执行 Bash 脚本，还能通过模型驱动自我迭代和编写工具，甚至借助扩展自动操控浏览器完成复杂的工作流。

2. 在 MacOS 安装 OpenClaw


硬件环境准备

设备类型：Mac（Intel 芯片或 Apple Silicon 芯片均可）。

内存建议：基础网关运行需至少 1GB RAM，建议 4GB 及以上以确保稳定的 NPM 构建和扩展运行。



软件要求

操作系统：macOS 12.0 或更高版本。

核心依赖：Node.js 22.0 或更高版本。

模型配置：需要提前准备好 Anthropic Claude、OpenAI 或 OpenRouter 的 API Key。



文件与权限配置

系统权限：确保你使用的终端（Terminal / iTerm2）在系统设置中拥有“完全磁盘访问权限”（Full Disk Access）。

目录结构：所有配置、密钥和状态均默认存储在 ~/.openclaw 目录下，沙盒工作区位于 ~/openclaw/workspace。



安装步骤

打开终端，安装或更新 Node.js 22（推荐使用 Homebrew）：brew install node@22。

输入 node --version 验证版本是否满足要求。

通过 NPM 全局安装 OpenClaw 最新版：npm install -g openclaw@latest。

运行初始化向导并注册后台守护进程：openclaw onboard --install-daemon。

在交互式向导中，选择你的大模型提供商，粘贴对应的 API Key，并配置你要接入的第一个通讯频道（如 Telegram Bot Token）。

运行 openclaw dashboard，在浏览器中打开 http://127.0.0.1:18789/ 进入可视化的控制面板。

3. 在 Windows 安装 OpenClaw

硬件环境准备

设备类型：支持虚拟化技术的 64 位 Windows PC。

内存建议：建议 8GB 及以上（主要用于为 WSL2 分配充足内存）。



软件要求

操作系统：Windows 10 或 Windows 11。强烈建议通过 WSL2（Windows Subsystem for Linux）进行隔离部署，而非直接使用原生 Windows 环境。

核心依赖：Node.js 22.0 或更高版本。



文件与权限配置

网络安全：Gateway 默认绑定 127.0.0.1:18789。切勿在未经授权的情况下将配置修改为暴露到公网（0.0.0.0），建议结合 Tailscale 或 SSH 隧道进行远程访问。

沙盒权限：若在 Docker / WSL 混合环境下遇到权限被拒（EACCES），需在宿主机修正挂载目录权限：sudo chown -R 1000:1000 ~/.openclaw。



安装步骤

以管理员身份打开 PowerShell，运行 wsl --install 安装 WSL2，然后重启电脑。

进入 Ubuntu/WSL 终端，运行环境配置命令：curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -。

安装 Node.js：运行 sudo apt-get install -y nodejs。

全局安装 OpenClaw：运行 npm install -g openclaw@latest。

初始化配置服务：执行 openclaw onboard --install-daemon，录入你的 API 密钥与偏好。

验证运行状态：输入 openclaw gateway status 检查后台服务是否成功拉起。

4. 安装常用技能 Skills

OpenClaw 强大的扩展能力来源于它的技能集（Skills）。你可以通过其内置的技能市场 Clawhub 快速赋能 Agent。

安装方法

交互式安装（推荐）：在你接入的聊天软件（如 Telegram）中，直接向 OpenClaw 发送自然语言指令，例如：“Please install a skill for me using Clawhub; its name is mcd.”。Agent 会自行联网拉取、配置并热重载该技能，无需你手动改代码。

手动配置：你可以通过修改 ~/.openclaw/TOOLS.md 和 AGENTS.md 文件来直接注入自定义的能力脚本。
